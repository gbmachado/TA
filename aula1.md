# Disponibilizando o CV via Surge

## Surge

O Surge é uma ferramenta para "static web publishing for Front-End Developers, right from the CLI".

Para instalá-lo:

```
$ npm install --global surge
```

> Para saber mais sobre o surge, acesse este link:
